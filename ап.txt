Лазерные оптико-акустические газоанализаторы имеют широкое применение в науке, экологии, промышленности. Такие устройства необходимы,
в частности для мониторинга загрязнения окружающей среды и в особенности атмосферы. Мониторинг необходим для оперативного контроля за
уровнем загрязнения окружающей среды для определения содержания загрязняющих веществ. Лазерный оптико-акустический газоанализатор (далее,
«ЛОАГ») позволяет с высокой точностью определить состав многокомпонентных лазерных смесей. В настоящее время ЛОАГ используется в составе
измерительного комплекса, который сопряжен с компьютером и оснащен специальным программным обеспечением. 
Отсутствие решения на Питон, позволяющего автоматизировать один из экспертных подходов к интерпретации результатов анализа при помощи ЛОАГ. 
На момент создания проекта существует ряд программных решений, способных автоматизировать различные подходы. Решение на Питоне ввиду относительной простоты и лаконичности данного языка программирования могло бы позволить эффективно автоматизировать один из подходов к интерпретации результатов анализа ЛОАГ.
Потенциальные потребители: экологические службы, государственные органы, медицинские учреждения. Касательно экологического значения уже упоминалось выше. По поводу использования в медицинских учреждениях, хотелось бы отметить следующее. Существует вероятность, что такое решение может быть востребовано в медицине для лечения и диагностики заболеваний органов дыхания).
Цель проекта:
Создать программный подход для автоматизации одного из экспертных подходов к интерпретации результатов ЛОАГ
